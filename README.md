## Description

Simple backend app for chat:
Users sending messages to each other. One user can be connected from multiple devices.
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## What using:

Nest.js,
TypeScript,
PostgreSQL,
TypeOrm,
Swagger,
WebSockets,
Unit & e2e testing examples (not much)

## Further development:

uploading/downloading files (streams),
Friendlists,
Group chats.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
